# 🗑️ Junkyard

![Storage](https://img.shields.io/badge/Storage-Assets-6366f1?style=for-the-badge&logo=icloud&logoColor=white)

A central repository for storing various assets and legacy code that I might need in the future.

### 📂 What's inside?
* **Assets:** Icons, images, and media resources used across different websites.
* **Backups:** Old projects, experiments, and discontinued code for archival purposes.
* **Resources:** Reusable snippets and design elements.

### ⚠️ Disclaimer
Everything here is provided **"as-is"**. Most of the code in the backups is no longer maintained and serves only as a personal reference.
